Model Architecture

State-of-the-art deep learning for face detection and recognition

1

Detection with VGG16

We employ the VGG16 convolutional neural network as our backbone for face detection. Using pre-trained weights and fine-tuning on our dataset, we achieve robust face detection capabilities. VGG16's deep architecture with small convolutional filters is particularly effective for feature extraction from facial images.

2

Recognition with Siamese Network

For face recognition, we utilize a Siamese network architecture that compares detected faces with stored images in our database. This network learns a similarity metric between faces, enabling accurate identification even with variations in lighting, angle, or facial expression.

Development Process

From data collection to trained model

1

Data Collection

We collected a comprehensive dataset of student and faculty images using OpenCV. The dataset includes diverse lighting conditions, angles, and facial expressions to ensure robust performance.

2

Image Annotation

Using LabelMe, we meticulously annotated each image to create accurate training data. This step was crucial for teaching the model what constitutes a face and how to differentiate between individuals.

3

Data Augmentation

We used the Albumentations library to create a more diverse dataset. Techniques included rotations, translations, lighting adjustments, and more to improve the model's generalization capabilities.
